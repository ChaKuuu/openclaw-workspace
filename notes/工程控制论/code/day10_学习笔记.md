## 1. 卡尔曼滤波要解决什么问题

**从带噪声的测量值中，估计真实状态。**

```
真实状态 x(t) ──系统──> 测量值 y = Cx + v(噪声)
     ↑
     └─── 卡尔曼估计 x_hat(t)
```

**核心矛盾：**
- 测量值有噪声 → 相信测量会受噪声影响
- 模型预测有误差 → 相信模型会被干扰影响
- 卡尔曼滤波 = 动态最优融合

---

## 2. 卡尔曼滤波方程

### 预测步（Prediction）
$$\dot{\hat{x}} = A\hat{x} + Bu$$
$$\dot{P} = AP + PA^T + Q$$

### 修正步（Correction）
$$K = P C^T R^{-1}$$
$$\hat{x} = \hat{x} + K(y - C\hat{x})$$
$$P = (I - KC)P$$

---

## 3. 符号说明

| 符号 | 含义 | 作用 |
|------|------|------|
| P | 估计误差协方差 | 估计的不确定性 |
| Q | 过程噪声协方差 | 模型误差来源 |
| R | 测量噪声协方差 | 传感器精度 |
| K | 卡尔曼增益 | 信任分配比例 |

---

## 4. 数值结果

$$A = -1, \quad B = 1, \quad C = 1$$
$$Q = 0.1, \quad R = 1.0$$

**稳态误差协方差：**
$$P_{ss} = 0.0488$$

**仿真结果：**
| 指标 | 数值 |
|------|------|
| 测量噪声标准差 | 0.977 |
| 估计误差标准差 | **0.130** |
| 噪声抑制比 | **7.5倍** |

---

## 5. 卡尔曼增益 K 的物理意义

$$K = \frac{P C^T}{R}$$

- **R 小（传感器好）** → K 大 → 更相信测量值
- **Q 大（过程噪声大）** → K 小 → 更相信模型预测
- **P 大（估计不确定）** → K 大 → 需要更多修正

---

## 6. 与 LQR 的关系

| | LQR | 卡尔曼滤波 |
|---|---|---|
| 目标 | 最小化 ∫(x'Qx + u'Ru)dt | 最小化 E[(x-x̂)'(x-x̂)] |
| 方程 | 连续 Riccati | 连续 Riccati |
| 结果 | 最优控制 u = -Kx | 最优估计 x̂ |

**合在一起 = LQG（Linear Quadratic Gaussian）控制**
- LQR 保证最优控制
- 卡尔曼滤波保证最优状态估计
