## 1. LQG = LQR + 卡尔曼滤波

LQG（Linear-Quadratic-Gaussian）是现代控制理论的核心组合：

```
LQR ─────> 最优控制 u = -K_lqr * x
  (状态反馈)        ↑
                   │ 估计状态
                   │
卡尔曼滤波 ──> x_hat = 估计状态
  (状态估计)
```

---

## 2. 分离原理（Separation Principle）

LQG 的关键洞察：**LQR 和卡尔曼滤波可以独立设计，自由组合。**

| 设计项 | 方法 | 独立？|
|------|------|------|
| LQR 最优控制 | 解 Riccati方程 | ✅ |
| 卡尔曼滤波 | 解 Riccati方程 | ✅ |
| 组合 | u = -K_lqr * x_hat | ✅ |

**稳定性保证：** LQG 组合后整体闭环系统稳定。

## 4. LQG 控制器的物理意义

**"盲人开车"问题：**
- 状态 x 不可直接观测（就像盲人看不见路）
- 只有测量 y 可用（就像盲人听声音）
- LQR 相当于"如果知道状态就最优控制"
- 卡尔曼滤波相当于"从测量中猜真实状态"
- LQG = 盲人开车 + 最优策略
