## 1. 线性 vs 非线性

| | 线性系统 | 非线性系统 |
|---|---|---|
| 叠加原理 | ✅ 成立 | ❌ 不成立 |
| 解的形式 | 闭式解 | 多无解析解 |
| 稳定性 | 特征值决定 | 复杂（极限环/分岔）|
| 典型方程 | ẋ = Ax | ẋ = f(x) |

---

## 2. 相平面（Phase Plane）

相平面 = (x, ẋ) 二维状态空间中的轨迹图。

**常见奇点类型：**

| 类型 | 特征 | 线性例子 |
|------|------|---------|
| 稳定节点 | 两特征值 < 0 | ẋ = -x, ẏ = -2y |
| 不稳定节点 | 两特征值 > 0 | ẋ = x |
| 鞍点 | 一正一负 | ẋ = x, ẏ = -y |
| 稳定焦点 | 复数，Re < 0 | ẋ = -0.5x - y, ẏ = x - 0.5y |
| 中心 | 纯虚数 | ẋ = -y, ẏ = x |

---

## 3. 极限环（Limit Cycle）

**自激振荡：孤立闭合轨道**

- 例子：van der Pol 振荡器
- 稳定极限环：附近轨道都收敛到它
- 不稳定极限环：扰动后发散
- 工程：时钟、PLL、心脏跳动

---

## 4. Lyapunov 间接法

**判断非线性系统在平衡点附近的稳定性：**

1. 求平衡点 x_e（满足 f(x_e) = 0）
2. 线性化：A = ∂f/∂x | x_e
3. 判据：
   - 所有 Re(λ) < 0 → 局部渐近稳定 ✅
   - 存在 Re(λ) > 0 → 不稳定 ❌
   - 纯虚 → 需要高阶项分析

**摆的线性化：**
```
ẋ₁ = x₂
ẋ₂ = -sin(x₁) - 0.5x₂

A = [[0, 1], [-cos(x₁), -0.5]] | (0,0)
  = [[0, 1], [-1, -0.5]]

λ = -0.25 ± 0.968i  → Re(λ) < 0 ✅
（小角度摆是稳定的）
```

---

## 5. 分岔（Bifurcation）

参数变化导致相平面拓扑结构突变：

- **鞍-结分岔**：节点/鞍点合并消失
- **Hopf 分岔**：平衡点失稳 → 极限环诞生
