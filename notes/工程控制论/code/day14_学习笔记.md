## 项目：卫星姿态控制

**物理模型：**
$$J\ddot{\theta} = u, \quad J = 1$$

**状态空间：**
$$x = \begin{bmatrix} \theta \\ \omega \end{bmatrix}, \quad A = \begin{bmatrix} 0 & 1 \\ 0 & 0 \end{bmatrix}, \quad B = \begin{bmatrix} 0 \\ 1 \end{bmatrix}$$

**测量：** 只测 θ（角度），不测 ω（角速度）

---

## 控制器设计步骤

### Step 1: 可控性检验 ✅
$$C_c = [B, AB] = \begin{bmatrix} 0 & 1 \\ 1 & 0 \end{bmatrix}, \quad \text{rank} = 2$$

### Step 2: LQR 最优控制
$$Q = \text{diag}(50, 10), \quad R = 0.1$$
$$K = [22.36, 12.03]$$
闭环极点：λ = [-2.30, -9.73]

### Step 3: 卡尔曼滤波器（观器）
$$Q_k = 0.1 \cdot I, \quad R_k = 5.0$$
$$L = [0.55, 0.14]$$
（只测角度，所以 ω 是估计的）

### Step 4: LQG 组合
$$u = -K \cdot \hat{x}$$
（控制器用估计状态，不是真实状态）

---

## 核心：分离原理

```
设计阶段：           运行阶段：
┌──────────────┐    ┌─────────────────────────┐
│  LQR: K      │    │ u = -K * x_hat         │
│  (独立设计)   │    │        ↑               │
└──────────────┘    │   估计状态 x_hat        │
┌──────────────┐    │        ↑               │
│ Kalman: L    │    │   y ──> 滤波器 ──> x_hat │
│  (独立设计)   │    └─────────────────────────┘
└──────────────┘
```
