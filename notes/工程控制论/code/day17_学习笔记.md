## 1. 直接法 vs 间接法

| | 间接法 | 直接法 |
|---|---|---|
| 方法 | 线性化 + 特征值 | 构造 V(x) |
| 适用范围 | 局部（小扰动）| 全局（可能）|
| 难度 | 简单 | 需要猜测 V(x) |
| 局限性 | 纯虚特征值失效 | V 函数不好找 |

---

## 2. Lyapunov 函数构造

**核心思想：** 找到"能量函数" V(x)

**条件：**

| 条件 | 含义 | 稳定性 |
|------|------|--------|
| V > 0, V̇ ≤ 0 | 能量不增加 | 稳定 |
| V > 0, V̇ < 0 | 能量持续减少 | 渐近稳定 |
| V > 0, V̇ > 0 | 能量增加 | 不稳定 |

---

## 3. 典型系统分析

**阻尼振子：**
$$V = \frac{1}{2}x^2 + \frac{1}{2}y^2 \quad \text{（总能量）}$$
$$\dot{V} = -cy^2 \leq 0 \Rightarrow \text{渐近稳定}$$

**van der Pol 振子：**
- 无阻尼（μ=0）：保守系统，能量守恒，闭合轨道（中心）
- 有阻尼（μ>0）：自激振荡，极限环（稳定的自我维持）

**摆（摩擦）：**
$$\dot{V} = -c \cdot \omega^2 \leq 0 \Rightarrow \text{渐近稳定到} \theta = 0$$

---

## 4. LaSalle 不变原理

如果 $\dot{V} \leq 0$ 但在某个非零点集合 $\dot{V} = 0$ 上系统不会停留（除原点外），则**渐近稳定**。

**核心：** "能量一直减少，最终会耗尽到零"
