## 1. H∞ vs LQG

| | LQG | H∞ |
|---|---|---|
| 噪声假设 | 高斯噪声（已知统计量）| 有界噪声（最坏情况）|
| 优化目标 | E[J] 最小化 | \|\|T_zw\|\|_∞ 最小化 |
| 设计思想 | 平均最优 | **最坏情况最优** |
| 鲁棒性 | 一般 | 强（对模型不确定性）|

---

## 2. H∞ 范数

$$\|T_{zw}\|_{\infty} = \sup_w \frac{\|z\|_2}{\|w\|_2} = \text{能量增益上界}$$

**物理含义：** 从扰动 w 到输出 z 的最大能量放大倍数

---

## 3. H∞ 控制问题

**目标：** 找到 K 使 \|\|T_zw\|\|_∞ 最小

$$\min_K \|T_{zw}\|_{\infty}$$

**约束条件：** $\gamma^2 > \|D\|^2$（确保有解）

**Riccati 方程：**
$$A^TP + PA + Q - P(BB^T - \frac{1}{\gamma^2}DD^T)P = 0$$

---

## 4. 鲁棒稳定性

$$\|\Delta\|_{\infty} < 1 + \|T_{zw}\|_{\infty} \leq \gamma \Rightarrow \text{鲁棒稳定}$$

**H∞ 保证闭环系统对"所有"有界扰动稳定，LQG 只保证对"统计平均"最优。**
