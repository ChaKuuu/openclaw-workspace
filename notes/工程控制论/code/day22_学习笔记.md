## 1. 可靠性基础

$$R(t) = e^{-\lambda t}$$

| 概念 | 公式 |
|------|------|
| 可靠度 | R(t) = P(系统时刻t正常) |
| 故障率 | λ = 1/MTBF |
| MTBF | 平均故障间隔时间 |

---

## 2. 冗余策略

**串联：** $R_s = R_1 \cdot R_2$（任一故障=系统故障）

**并联：** $R_p = 1 - (1-R_1)(1-R_2)$（全部故障=系统故障）

**2取1（2oo3）：** $R = 3R^2 - 2R^3$（容忍1个故障）

---

## 3. 关键洞察

- **冗余是安全的代价**：成本更高，但更安全
- **MTBF 并联 > 串联**：并联系统的平均寿命更长
- **投票系统在航空航天中普遍使用**（三重冗余TMR）
