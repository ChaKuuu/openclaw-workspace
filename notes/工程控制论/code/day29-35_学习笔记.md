## Day 29: Poincare 截面 + 同/异宿轨道

**Poincare截面：**
- 在驱动周期 T_drive 采样状态 (n-dim → (n-1)-dim)
- 周期轨道 → 离散点
- 准周期 → 闭合曲线
- 混沌 → 分形散点

**同宿轨道（Separatrix）：**
- 连接鞍点到自身的轨道
- 能量 E=2（无阻尼摆）
- 扰动 → 同宿纠缠 → 混沌的门户

## Day 30: 中心流形 + Normal Forms

**中心流形理论：**
- 慢变方向（特征值在虚轴）主导长期行为
- 快方向指数衰减

**Normal Forms（规范形式）：**
| 分岔 | Normal Form |
|------|------------|
| 鞍结 | $x_dot = s + x^2$ |
| 跨临界 | $x_dot = \mu x - x^2$ |
| 叉式 | $x_dot = \mu x - x^3$ |
| Hopf | $r_dot = \mu r - r^3$ |

## Day 31: Hopf 分岔

**类型：**
- **超临界 Hopf**（$\mu > 0$）：稳定平衡 → 稳定极限环（软过渡）
- **亚临界 Hopf**（$\mu < 0$）：不稳定极限环收缩 → 突变（滞回）

**判据：**
- $\text{trace}(A) = 0$ 经过虚轴
- $\det(A) > 0$（一对复特征值）

## Day 32: 混沌控制 (OGY)

**OGY 方法（Ott-Grebogi-Yorke, 1990）：**
1. 识别期望的周期轨道（混沌中有无穷多 UPO）
2. 线性化：$\delta x_{n+1} = A \delta x_n + B \delta p$
3. 计算所需小扰动 $\delta p = -(1-A)/B \cdot \delta x_n$
4. 施加噪声级扰动即可稳定

**Pyragas（TFD）：**
$$u(t) = K[y(t-\tau) - y(t)]$$
- $\tau$ = 不稳定轨道周期
- OGY 的连续版本

## Day 33: Lorenz 系统

**方程：**
$$x' = \sigma(y-x), \quad y' = x(\rho-z)-y, \quad z' = xy - \beta z$$

**分岔行为：**
- $\rho < 1$：稳定原点
- $1 < \rho < 24.74$：两个稳定焦点
- $\rho > 24.74$：**混沌**

**蝴蝶效应：** 初始条件微小差异 → 指数级分离（LE > 0）

## Day 34: 分形维度

**维度类型：**
- Hausdorff：$D_H = \lim_{\epsilon \to 0} \frac{\log N(\epsilon)}{\log(1/\epsilon)}$
- Box-Counting：同上
- Kaplan-Yorke：$D_{KY} = k + \frac{\sum_{i=1}^k \lambda_i}{|\lambda_{k+1}|}$

**Lorenz (ρ=28)：**
- $\lambda = [0.906, 0, -14.572]$
- $D_{KY} = 2 + 0.906/14.572 = 2.062$
