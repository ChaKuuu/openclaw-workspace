# Day 36 - Pontryagin 极大值原理

## 核心概念

### 三件套
1. **哈密顿函数**: H = L(x,u) + λᵀf(x,u)
2. **协态方程**: λ̇ = -∂H/∂x
3. **最优性条件**: ∂H/∂u = 0

### 与 LQR 的关系
- LQR 是 PMP 的特例（线性系统 + 二次型代价）
- PMP 是 LQR 的父类（适用于任意系统）

## 最速降线问题（经典例子）

**问题**: 找出曲线使质点最快从 A 滑到 B

**求解**:
```
H = 1 + λ₁v + λ₂y
λ̇₁ = 0 → λ₁ = const
∂H/∂θ = λ₁cosθ = 0 → θ = ±π/2
```

**答案**: 摆线 (cycloid)

## 倒立摆仿真实战

### 现象
- 大初始角度(45°)下，PD控制和简化PMP都失败
- BVP求解遇到奇异雅可比

### 原因
- 非线性程度太高
- 初始猜测远离真解

### 解决方向
1. 线性化 → 用 LQR
2. 梯度下降 → 数值优化
3. 更好的初始猜测 → 多重射击法

## 工程意义

**PMP 提供了一般性框架，但实际工程中：**
- 简单系统 → 解析求解
- 复杂非线性 → 用 LQR 近似或数值优化
- 工业首选 → 依然是 PID 和 LQR
