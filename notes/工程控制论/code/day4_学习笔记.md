## 1. 频率响应基础

对传递函数 G(s)，令 s = jω：

$$G(j\omega) = |G(j\omega)| \cdot e^{j\angle G(j\omega)}$$

- **幅频特性**：|G(jω)| 随 ω 的变化
- **相频特性**：∠G(jω) 随 ω 的变化

**Bode 图** = 幅频 + 相频，分别画在对数坐标上。

---

## 2. 典型环节的 Bode 图

### 惯性环节 G(s) = K/(Ts+1)

**幅值：**
$$|G(j\omega)| = \frac{K}{\sqrt{1+(\omega T)^2}}$$

**对数幅值（dB）：**
$$20\log|G(j\omega)| = 20\log K - 20\log\sqrt{1+(\omega T)^2}$$

**折线近似：**
- 低频（ω << 1/T）：20log K（水平线）
- 高频（ω >> 1/T）：斜率 -20dB/dec 的直线
- **转折频率** ω_c = 1/T

**相角：**
$$\angle G(j\omega) = -\arctan(\omega T)$$

- ω → 0：相角 0°
- ω = 1/T：相角 -45°
- ω → ∞：相角 -90°

## 3. 谐振峰值（二阶系统）

对于欠阻尼二阶系统：
$$M_r = \frac{1}{2\zeta\sqrt{1-\zeta^2}}$$

| ζ | M_r | 特征 |
|---|---|---|
| 0.1 | 5.03 | 强烈谐振 |
| 0.3 | 1.75 | 明显谐振（**4.85 dB**）|
| 0.5 | 1.15 | 轻微谐振 |
| 0.707 | 1.00 | 无谐振（Butterworth）|
| >= 0.707 | 1.00 | 无谐振 |

**谐振频率：** ω_p = ω_n√(1-2ζ²)

## 5. Bode 图工程应用

### 带宽（Bandwidth）
系统响应速度的上限频率。
- 带宽大 → 响应快（高频信号能通过）
- 带宽小 → 响应慢（只能跟踪缓慢变化）

### 稳定裕度
从 Bode 图可以读出：
- **相位裕度**（Phase Margin）：在增益交叉频率处，相角距 -180° 的距离
- **增益裕度**（Gain Margin）：在相角 -180° 频率处，增益距 0 dB 的距离

### 工程意义
> 增益裕度 > 6 dB + 相位裕度 > 45° = 稳健系统

4. **逐段叠加**：每个转折频率改变斜率
5. **校验**：在关键点验证 dB 值
