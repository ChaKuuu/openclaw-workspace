## 1. Nyquist 判据核心公式

$$Z = N + P$$

| 符号 | 含义 |
|------|------|
| **Z** | 闭环右半平面极点数（Z=0 则稳定）|
| **N** | Nyquist 曲线绕 -1 点的**逆时针**圈数 |
| **P** | 开环传递函数右半平面极点数 |

**稳定条件：Z = 0**

---

## 2. 如何画 Nyquist 曲线

对 G(s)：
1. 令 s = jω，ω 从 0 → ∞，画出 G(jω) 曲线
2. ω 从 -∞ → 0，取对称（共轭对称）
3. 补上从 ω=∞ 绕到 ω=j∞ 的大半圆

**简化规则**（适用于物理可实现系统）：
- 正频率部分画出后，负频率部分取镜像

## 3. 三种情况分析

### 情况 1：G(s) = 1/(s+1)（稳定）
```
P = 0（开环无 RHP 极点）
N = 0（不包围 -1 点）
Z = 0 + 0 = 0 → 稳定 ✅
```

### 情况 2：G(s) = 1/(s-1)（不稳定）
```
P = 1（开环有 1 个 RHP 极点）
N = -1（顺时针包围 -1 一次）
Z = -1 + 1 = 0 → 看起来稳定
  但 -1 点在曲线上 → 临界稳定/不稳定！
```

### 情况 3：G(s) = K/(s-1)，K=2
```
P = 1
N = -1（顺时针包围）
Z = -1 + 1 = 0
  但 -1 在外部更多 → 不稳定
```

---

## 4. 混参系统：G(s) = 1/((s+1)(s+2)(s-0.5))

```
P = 1（s-0.5 在 RHP）
N ≈ 1（逆时针包围 -1 一次）
Z = 1 + 1 = 2 → 不稳定（2个闭环 RHP 极点）❌
```

## 5. Nyquist 与 Bode 的关系

| 判据 | 优点 | 缺点 |
|------|------|------|
| **Nyquist** | 一眼看稳定性，了解相位裕度 | 需要复平面作图 |
| **Bode** | 实验数据可直接画图 | 需分别看增益/相位裕度 |
| **Routh** | 直接代数计算 | 高阶系统计算繁琐 |

**三者互相验证：** Routh 第一列法 = Nyquist 的简化版

## 6. Python 验证

```python
from scipy import signal
import numpy as np

# 稳定系统
sys = signal.TransferFunction([1], [1, 1])
poles = sys.poles
print(f"Poles: {poles}")
print(f"RHP poles: {sum(1 for p in poles if p.real > 0)}")

# Nyquist 曲线
w = np.logspace(-2, 2, 500)
_, H = signal.freqresp(sys, w)
```

## 7. 稳定裕度的 Nyquist 解读

- **相位裕度**：在增益交叉频率（|G|=1）处，相角距 -180° 的距离
- **增益裕度**：在相角 -180° 频率处，增益距 0 dB 的距离

> **在 Nyquist 图上**：-1 点越"远"，裕度越大
