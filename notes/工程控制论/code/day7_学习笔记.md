## 核心公式速查

```
惯性环节:        y = K(1 - e^(-t/T))
二阶欠阻尼:      Mp = exp(-πζ/√(1-ζ²))
                 tp = π/(ωn√(1-ζ²))
                 ts = 4/(ζωn)  (2%准则)
Routh:           第一列同号 → 稳定
Nyquist:         Z = N + P
PID:             C(s) = Kp + Ki/s + Kd·s
Z-N整定:         Kp = 1.2T/(Kθ)
```
