# 第2篇：线性系统理论

---

## 概述：为什么从线性系统开始？

非线性系统的数学分析极其困难。钱老的策略是：**先精通线性系统，打好基础，再逐步处理非线性。**

> "线性系统是理想化模型，但它是理解真实系统的起点和基准。"

---

## 第4章 线性微分方程与传递函数

### 传递函数：系统的"指纹"

**定义：** 在零初始条件下，输出的拉普拉斯变换除以输入的拉普拉斯变换。

```
G(s) = Y(s) / U(s)
s = σ + jω（复频率）
```

**为什么重要：**
- 系统的动态特性一目了然
- 零极点分布 → 直观判断稳定性
- 串联系统的总传递函数 = 各子系统传递函数的乘积

### 典型环节的传递函数

| 环节类型 | 传递函数 | 时域响应特征 |
|---------|---------|------------|
| 比例环节 | K | 无延迟，无惯性 |
| 惯性环节（一阶） | K/(Ts+1) | 指数上升/下降 |
| 振荡环节（二阶） | K/(T²s²+2ζTs+1) | 振荡收敛/发散 |
| 微分环节 | K·s | 提前预测趋势 |
| 积分环节 | K/s | 累积误差消除 |

### 极点-零点与系统动态

- **极点位置决定响应速度**：越靠左（负实部越大）→ 响应越快
- **极点位置决定振荡性**：共轭复极点 → 振荡响应
- **零点影响响应形状**：不影响稳定性，但影响超调量

### 进化路线应用
- "传递函数" = 你的**核心能力结构**
- 了解自己能力的时间常数T：多久能看到成效？
- 有没有振荡环节：技能提升是否忽高忽低不稳定？
- 零点：某些输入不会直接影响输出（如学历≠实际能力）

## 第5章 频率响应法

### Bode图：工程设计的利器

**对数频率特性图（伯德图）：**
- 幅值图：|G(jω)| dB vs log ω
- 相位图：∠G(jω) vs log ω

### 稳定裕度：系统安全的"护栏"

**增益裕度（Gain Margin）：**
- 相位=-180°时，幅值|G(jω)|距离0dB的余量
- 工程要求：GM ≥ 6dB（越大越安全）

**相位裕度（Phase Margin）：**
- 幅值=0dB时，相位距离-180°的余量
- 工程要求：PM ≥ 45°（越大越稳定）

```
Bode图中的稳定裕度示意：

       幅值(dB)
         │
    0dB ─┼─────────────────── 交界频率( crossover)
         │         \
         │          \___
         │              \___     -180°线
         │                  \___
    -GM ─┤                      \___
         │                          \___
         └───────────────────────────────→ ω
              PM: 在0dB处与-180°线的距离
```

### 频域性能指标

| 指标 | 物理意义 | 进化路线对应 |
|------|---------|------------|
| 带宽( bandwidth) | 系统响应速度上限 | 适应新环境的最大速度 |
| 共振峰值(Mr) | 对特定频率的放大程度 | 对某类刺激的敏感度/放大效应 |
| 低频增益 | 稳态跟踪精度 | 长期目标的坚持能力 |

### 奈奎斯特稳定判据
- 利用开环传递函数G(s)H(s)的奈奎斯特图判断闭环稳定性
- **优点**：不需要解特征方程，适用于时滞系统

### 进化路线应用
- **带宽概念**：我的学习带宽/适应带宽有多大？
- **增益裕度**：面对压力能承受多少额外负荷而不崩溃？
- **相位裕度**：反馈延迟有多大？（复盘周期太长？）
- 💡 行动：测量自己的"频率响应"——单位时间能处理多少变化？

## 第6章 根轨迹法

### 根轨迹的基本原理

**闭环极点**是开环传递函数G(s)H(s)的极点与零点共同决定的。

**根轨迹作图规则：**
1. 起点：开环极点；终点：开环有限零点（或无穷）
2. 分支数 = 极点数
3. 渐近线角度和交点有公式可循

### 根轨迹的设计意义

通过**添加零点、极点**来"塑造"闭环极点位置：
- 在左半平面 → 稳定且快速
- 靠近原点 → 缓慢但稳定
- 穿越虚轴 → 临界稳定/振荡

### 进化路线应用
- "添加零点" = 引入**前馈/补偿**，改善响应
- "添加极点" = 引入**惯性/延迟**，减缓响应
- 根轨迹越靠近虚轴 → 系统越接近不稳定边缘
- 💡 关键问题：我的"闭环极点"在哪里？离不稳定边界有多远？

## 本篇总结：线性系统的工程思维

| 概念 | 工程含义 | 进化应用 |
|------|---------|--------|
| 传递函数 | 系统的输入-输出特性 | 能力结构/影响力结构 |
| 频率响应 | 对不同变化速率的响应能力 | 快速适应 vs 稳步积累 |
| Bode图 | 可视化稳定性裕度 | 安全边际/抗压能力 |
| 根轨迹 | 调整参数的极点移动规律 | 策略调整的预期效果 |
