## 第3篇：控制系统设计

### 第7章 调节原理（PID控制器）

**PID是工业控制皇冠上的明珠——历经百年仍未褪色。**

**三种控制作用的分工：**

```
PID输出 u(t) = Kp·e(t) + Ki·∫e(t)dt + Kd·de(t)/dt
            = 比例项    +  积分项     +   微分项
```

| 环节 | 作用 | 类比进化场景 |
|------|------|------------|
| P（比例） | 偏差的直接响应，有偏差就有控制 | 问题出现→直接行动纠正 |
| I（积分） | 消除稳态误差，累积偏差→持续调整 | 长期存在的问题需要持续投入 |
| D（微分） | 预测趋势，提前干预 | 见微知著，防患于未然 |

**PID参数调节口诀（Ziegler-Nichols）：**
- 先只调P，直到持续振荡
- 记录临界增益Ku和临界周期Pu
- 按经验公式整定：Kp=0.6Ku，Ki=1.2Ku/Pu，Kd=3KuPu/40

**PID的局限：**
- 非线性系统、强干扰 → 效果差
- 参数固定，无法适应过程变化
- 需要进阶到自适应控制（见第6篇）

### 第8章 串联校正与并联校正

**串联校正（串联补偿器）：**
- 插在误差信号和执行器之间
- Lead校正：提升相位裕度，加快响应
- Lag校正：提升低频增益，提高稳态精度
- Lead-Lag：兼顾动态和稳态

**并联校正（反馈校正）：**
- 在系统内部建立附加反馈
- 速度反馈：抑制非线性影响
- 压力反馈：改变系统阻尼特性

**设计哲学：** 
> "能用简单的就不引入复杂的。能用串联的就不必并联。"

### 第9章 继电（Bang-Bang）控制

**最简单的最优控制：只有两个控制量，上限或下限。**

- 例子：开关灯、火箭推进器全开/全关
- 最优性：时间最优（最快达到目标）
- 代价：能耗大、冲击大、频繁切换

## 第4篇：最优控制

### 第10章 变分法与庞特里亚金极小值原理

**变分法基础：**
- 泛函J[u] = ∫L(x, u, t)dt → 求u(t)使J最小
- 欧拉-拉格朗日方程 → 泛函极值的必要条件

**庞特里亚金极小值原理（1956）：**

```
哈密顿函数：H = L + λᵀf(x, u)

最优控制 u*(t) 使 H 在每时刻取最小值：
u*(t) = arg min_{u∈U} H(x*(t), u, λ*(t), t)
```

**物理意义：**
> 在所有可能的控制中，选择使"代价 + 伴随状态×系统动态"最小的那个。

**控制约束的处理：**
- |u| ≤ u_max（执行器有物理限制）
- 极小值原理天然处理这类约束，比变分法更强大

### 第11章 动态规划

**贝尔曼最优性原理：**
> "一个最优策略的子策略，对于该子策略的初始状态来说，仍然是最优的。"

**贝尔曼方程：**
```
V(x, t) = min_{u∈U} { L(x, u, t) + V(f(x, u, t), t+Δt) }
```

**维度诅咒：**
- 状态空间维度n → 计算量指数增长
- n=10以上就难以精确求解
- → 近似动态规划、神经网络近似 Q函数

### 第12章 线性二次型调节器（LQR）

**最优雅的最优控制形式：**

- 线性系统：dx/dt = Ax + Bu
- 二次型性能指标：J = ∫(xᵀQx + uᵀRu)dt
- 求解：黎卡提代数方程 → 反馈矩阵K → u = -Kx

**工程中的LQR：**
- 姿态控制：x=角度偏差，u=推力指令
- Q权重：状态偏差的容忍度
- R权重：控制能量的消耗代价

**LQR的哲学意义：**
> "最优控制不是无限追求性能，而是权衡性能与代价。"

## 本两篇的进化路线应用

### PID思维 → 自我调节的四种模式

| PID模式 | 控制论 | 个人进化 |
|---------|-------|--------|
| P模式 | 有偏差立即行动 | 应急反应式成长，见招拆招 |
| PI模式 | 持续消除稳态误差 | 长期问题持续投入，累积效果 |
| PD模式 | 预测趋势提前干预 | 见微知著，在问题爆发前解决 |
| PID | 完整控制 | 综合策略：即时+持续+预防 |

### 最优控制的三大约束 → 个人进化的时间管理

| 最优控制约束 | 含义 | 个人进化应用 |
|------------|------|------------|
| 控制幅值约束 | 执行器有最大输出 | 每天精力有限，不能无限投入 |
| 燃料/能量约束 | 总资源有限 | 人生精力总量有限，需要战略配置 |
| 时间最优 | 尽快达到目标 | 有时限的任务：Bang-Bang控制 |
| 能量最优 | 用最少资源达到目标 | 长期项目：平滑控制，均匀用力 |

### 动态系统视角看个人成长

```
d(能力)/dt = f(当前能力, 当前努力, 环境) + w(t)（噪声）

→ 状态方程
→ 存在最优努力轨迹
→ 过度努力（u过大）→ 系统振荡
→ 努力不足（u过小）→ 收敛太慢或停滞
```

💡 **核心洞察：成长是最优控制问题，不是线性累积问题。**
