## 第5篇：随机系统与滤波

### 第13章 随机过程基础

**随机过程是现实工程系统的真实面貌——确定性模型只是近似。**

**关键概念：**
- **白噪声**：功率谱密度均匀，任意时刻不相关（理想化模型）
- **相关函数**：R(τ) = E[x(t)x(t+τ)] → 描述时间相关性
- **功率谱密度**：傅里叶变换 → 频率域的能量分布
- **平稳过程**：统计特性不随时间变化（ ergodic 可遍历）

### 第14章 维纳滤波

**维纳滤波的目标：在最小均方误差准则下，从噪声污染的观测中恢复真实信号。**

**核心方程（维纳-霍普夫方程）：**
```
R_xy(τ) = ∫R_x(τ-σ)h(σ)dσ  (连续)
R_xy(k) = ΣR_x(k-i)h(i)      (离散)
```

**维纳滤波的局限：**
1. 只适用于平稳随机过程
2. 需要无限过去的观测数据
3. 计算复杂度高，难以实时实现

### 第15章 卡尔曼滤波（系统状态估计的核心）

**卡尔曼滤波 = 维纳滤波的递归实现 + 状态空间模型**

**五大公式：**

```
预测阶段：
x̂(k|k-1) = A·x̂(k-1|k-1) + B·u(k)        （状态预测）
P(k|k-1)  = A·P(k-1|k-1)·Aᵀ + Q            （预测误差协方差）

更新阶段：
K(k)  = P(k|k-1)·Hᵀ·(H·P(k|k-1)·Hᵀ + R)⁻¹ （卡尔曼增益）
x̂(k|k) = x̂(k|k-1) + K(k)·(z(k) - H·x̂(k|k-1))  （状态更新）
P(k|k)  = (I - K(k)·H)·P(k|k-1)             （更新误差协方差）
```

**增益K(k)的物理意义：**
- R大（观测噪声大）→ K小 → 更相信模型预测
- P大（预测不确定）→ K大 → 更相信新的观测
- 自动权衡：观测 vs 模型的信任度

**"预测-校正"框架的哲学意义：**

```
维纳滤波：全局最优，但需要所有历史数据
卡尔曼滤波：递推最优，每步只用到上一步结果
→ 实时性 ✓
→ 非平稳过程 ✓
→ 工程可实现 ✓
```

**阿波罗登月的核心算法之一。钱老在同时代推动了这一理论在航天中的应用。**

## 第6篇：非线性与自适应控制

### 第16章 相平面分析

**相平面：状态x vs 速度dx/dt 的二维平面**

非线性系统的相轨迹可视化：
- **稳定焦点**：阻尼振荡收敛 → 渐进稳定
- **不稳定焦点**：发散振荡 → 不稳定
- **中心点**：等幅振荡（保守系统）
- **鞍点**：不稳定，表现为"分水岭"
- **极限环**：闭合轨迹 → 自激振荡（与初始条件无关的固有频率）

**工程意义：** 极限环是非线性系统特有的现象，正反馈+延迟导致。

### 第17章 描述函数法

**用等效线性增益近似非线性元件的频率响应。**

假设非线性元件输入为正弦信号，输出基波分量即为描述函数N(A)：
- N(A) 是输入振幅A的函数（非线性特征）
- 可以用奈奎斯特判据分析包含非线性的闭环系统

**典型应用：** 判定自激振荡（极限环）的存在性和稳定性。

### 第18章 李雅普诺夫稳定性理论

**稳定性是控制系统的第一要求——不稳定系统是危险的。**

**第一方法（间接法）：**
- 将非线性系统在平衡点附近线性化
- 根据雅可比矩阵特征值判断局部稳定性
- 局限：只适用于"附近"，无法处理大偏差

**第二方法（直接法，核心）：**
- 构造李雅普诺夫函数V(x)：类似"能量"的正定标量
- V̇(x) < 0 → 系统渐近稳定（无需解方程！）
- 物理直觉：能量持续耗散 → 系统最终停在最小能量点

```
李雅普诺夫稳定性判定框架：

V(x) > 0  (正定)
V̇(x) = dV/dt < 0  (负定)  →  渐近稳定
V(x) → ∞ 当 ||x|| → ∞  →  全局渐近稳定
```

**对非线性控制理论的深远影响：**
- 无需知道精确模型即可判定稳定性
- 为鲁棒控制提供理论基础
- 现代控制理论的基石之一

### 第19章 自适应控制初步

**模型参考自适应控制（MRAC）：**

```
参考模型：指定期望的输出响应
          xm(t) = Am·xm + bm·r(t)

实际系统：可调参数控制器
          x(t) = A·x + B·u(θ)

自适应律：调整控制器参数θ使误差 e = x - xm → 0
```

**核心思想：边飞边造翅膀**

- 不能等到完全了解系统再去控制
- 在控制过程中持续学习/调整
- 适用于系统参数缓慢变化的情况

**自整定控制器：**
1. 在线参数辨识：估计被控对象参数
2. 根据辨识结果重新设计控制器参数
3. 比MRAC更直接，但计算量更大

**自适应 vs 鲁棒：**
- 自适应：主动调整参数适应变化
- 鲁棒：设计固定参数应对最坏情况

## 本篇的进化路线应用

### 卡尔曼滤波 → 认知自我

```
真实状态 x(t)  →  我实际是什么样的人
观测 z(t)      →  外界的反馈/评价
模型预测       →  我对自己的预期
卡尔曼滤波     →  综合真实状态的最佳估计

关键问题：
- Q（过程噪声）：我的行动有多少不确定性？
- R（观测噪声）：外界评价有多少噪声/偏见？
- 卡尔曼增益太高：太相信外部评价，被噪声左右
- 卡尔曼增益太低：太相信自己的模型，忽视反馈
```

### 李雅普诺夫 → 学习节奏

```
V(x) = 能量函数
V̇(x) < 0 = 能量持续减少

学习中的"能量函数"：
- V = 焦虑水平 + 愧疚感 + 低效能感
- 健康的成长：V递减，但不是负值（负值=麻木）
- 崩溃/倦怠：V̇突然变正（能量不降反升）
```

### 自适应控制 → 成长策略

```
MRAC思维：
- 有一个"参考模型"（你想成为什么样的人）
- 持续比较你和参考模型的差距
- 调整策略使差距减小

自整定思维：
- 每季度重新评估"被控对象模型"（你自己变了多少？）
- 外部环境变了多少？
- 控制器参数需要调整吗？
```

## 关键洞察

> **工程控制论的核心矛盾：**
> - 精确性 ↔ 实时性
> - 最优性 ↔ 计算复杂度
> - 理论完美 ↔ 工程实现

> **进化的核心矛盾：**
> - 想快速成长 ↔ 资源有限
> - 完美主义 ↔ 迭代速度
> - 长期目标 ↔ 短期压力
